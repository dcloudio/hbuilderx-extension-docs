# Solution for uniapp Publishing Server Response Exception

### Error Description

When publishing uniapp to WeChat Mini Program, the error message appears: `Server response exception, please check local network status or contact customer service`

<img src="https://web-ext-storage.dcloud.net.cn/hx/faq/uniapp_publish_server_error.jpg" style="zoom: 60%;" />

### Solution

Download the SSL certificate below and install it on your computer.

Certificate download link: [https://web-ext-storage.dcloud.net.cn/hx/questions/certum.pem](https://web-ext-storage.dcloud.net.cn/hx/questions/certum.pem)
