# CLI uni-app 运行日志 - uniCloud

> HBuilderX cli 命令行工具

> 需HBuilderX 4.87+ 版本

通过 CLI 查看运行到 uniCloud 的日志。

## 命令说明

### logcat unicloud

查看运行到 uniCloud 日志

**用法：**

```shell
./cli logcat unicloud --project 项目名称
```

**参数：**

| 参数名称  | 描述                                  |
| --------- | ------------------------------------- |
| --project | HBuilder X 里导入的项目名称或绝对路径 |

**使用示例：**

```shell
# 查看uniCloud日志
./cli logcat unicloud --project 项目名称
```
