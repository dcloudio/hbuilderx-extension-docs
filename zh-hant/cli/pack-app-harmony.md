# CLI uni-app 發行 - App-Harmony-本地打包@packAppHarmony

> HBuilderX cli 命令行工具

通過 CLI 對鴻蒙平臺進行本地打包，支持 uni-app (x) 項目。

## 命令說明

### pack app-harmony

對鴻蒙平臺進行本地打包

**參數：**

| 參數      | 說明           |
| --------- | -------------- |
| --help    | 顯示命令幫助   |
| --project | 必填，項目名稱 |

**使用示例：**

```shell
# 查看命令幫助
cli pack app-harmony --help

# 對鴻蒙平臺進行本地打包
cli pack app-harmony --project 項目名稱
```

**注意事項：**

> 首先, 需要啓動 HBuilderX. (進入 HBuilderX 安裝目錄根目錄, 終端輸入 `cli.exe open`)
