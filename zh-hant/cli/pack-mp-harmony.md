# CLI uni-app 發行 - 鴻蒙元服務打包@packMpHarmony

> HBuilderX cli 命令行工具

通過 CLI 對鴻蒙元服務平臺進行本地打包，支持 uni-app 項目。

## 命令說明

### pack mp-harmony

對鴻蒙元服務平臺進行本地打包

**參數：**

| 參數      | 說明           |
| --------- | -------------- |
| --help    | 顯示命令幫助   |
| --project | 必填，項目名稱 |

**使用示例：**

```shell
# 查看命令幫助
cli pack mp-harmony --help

# 對鴻蒙元服務平臺進行本地打包
cli pack mp-harmony --project 項目名稱
```

**注意事項：**

> 首先, 需要啓動 HBuilderX. (進入 HBuilderX 安裝目錄根目錄, 終端輸入 `cli.exe open`)
